# StyleMate 👗

A smart outfit generator that creates outfit combinations from your wardrobe.

## 🔥 Features

* Upload and store clothes
* Generate outfits instantly
* Color harmony scoring system
* Explanation for each outfit
* Save favorite outfits
* Works fully offline

## 🧠 How It Works

* Clothes are stored in localStorage
* Outfit is generated using category filtering + randomization
* Color harmony score determines quality
* Explanation helps user understand styling

## 🛠 Tech Stack

* HTML
* CSS
* JavaScript
